## ✨ Features

- Enter numbers and operators (`+`, `-`, `*`, `/`, `.`)
- Display the expression on the screen
- Calculate the result by clicking "="
- Clear the expression with the "C" button
- Show error message for invalid expressions



## 🛠️ Technologies Used

- HTML5
- CSS3 
- JavaScript
